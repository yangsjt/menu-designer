# 动径 MovePath — 产品需求文档 (PRD)

## 产品概述

**产品名称：** 动径 MovePath 店铺菜单板

**产品定位：** 动径 MovePath 是"梧桐区街区探索站"，融合咖啡与骑行文化的城市空间。菜单板为店内展示用横版大屏/印刷菜单，向顾客传达完整的产品线和品牌调性。

**核心诉求：**
- 一目了然展示全部饮品与租赁服务
- 传递骑行 × 咖啡的品牌特色
- 支持直接印刷输出（125cm × 60cm）

## 各区域功能说明

### A. Header (66px)

- 品牌 Logo（SVG 白色版，117px 宽）
- 品牌标语：`咖啡与骑行的城市探索站`
- 英文副标语：`URBAN EXPLORATION HUB OF COFFEE & CYCLING`

### B. Col 1 — 季节限定 (290px)

- 橙色标题栏（26px 高）
- 3 张全出血图片卡片，等高分布
- 每张卡片：产品图 + 底部渐变蒙版 + 品名/英文名/描述/价格
- 高价商品（>50）添加橙色轮廓线（`.premium`）

### C. Col 2 — 动径特调 + 能量组合 (430px)

**动径特调：**
- 2×4 网格布局，7 款特调 + 1 格 Logo 水印
- 每格：中文名、英文名、价格、描述
- 高价商品（>38）添加橙色渐变背景（`.accent`）

**能量组合：**
- 3 张迷你横卡，左侧橙色边框
- "减"类型：深色背景，badge 显示"减N"
- 固定价格类型：橙色背景（`.highlight`），badge 显示数字

### D. Col 3 — 经典日常 + 非咖啡 + 自行车租赁

**经典日常（左子列）：**
- 点线价目表格式：品名 ··· 价格
- 6 款基础咖啡 + 2 个升级项
- 升级项缩进显示，价格带 `+` 前缀

**非咖啡类（左子列续）：**
- 同样点线价目表格式
- 6 款非咖啡饮品

**自行车租赁（右子列）：**
- 产品图 + 渐变蒙版叠加
- 主价格突出显示（15 分钟计费）
- 三档时长价格（1H / 4H / 24H）
- 租赁流程说明（扫码 > 押金 > 出发）

### E. Footer (26px)

- 三项优惠信息：骑行到店减3、自带杯减3、宠物友好
- 竖线分隔，配 emoji 图标
- 右侧货币单位说明

## 数据源与更新流程

### 数据源文件

`docs/动径咖啡店铺菜单 - 当前菜单.csv`

这是菜单的**唯一数据源**。所有菜品信息的变更应先更新 CSV，再同步到 HTML。

### 更新流程

1. 编辑 CSV 文件（修改价格、品名、描述等）
2. 参照 `CLAUDE.md` 中的 CSV → HTML 映射表
3. 定位 HTML 中对应元素并修改
4. 浏览器预览验证显示效果

### CSV 字段说明

| 列 | 字段名 | 类型 | 说明 |
|----|--------|------|------|
| B | 序号 | 数字 | 1–31 顺序编号 |
| C | 类别 | 文本 | 6 个类别，格式 `N. 类别名` |
| D | 品名 | 文本 | 中文产品名称 |
| E | 英文名/包含内容 | 文本 | 英文名或套餐内容 |
| F | 描述/规格 | 文本 | 产品描述或规格说明 |
| G | 价格 (¥) | 数字/文本 | 纯数字或 `减 N` 格式 |

## 设计约束

### 尺寸规范

详见 `README.md` 中的完整设计规范，核心约束：

- Header 高度：66px
- Footer 高度：26px
- Section Header：普通 22px / 橙色 26px
- 季节限定图片：16:9 比例

### 色彩系统

参照 `README.md` 色彩系统章节，主要 CSS 变量定义在 `:root` 中。

### 字体

| 字体 | 用途 |
|------|------|
| Noto Sans SC | 中文正文（300/400/500/700） |
| Bebas Neue | 英文标题、价格、标签 |

### 价格显示

- 全部使用纯数字，不添加任何货币符号（"元"、"¥"）
- 升级项价格加 `+` 前缀
- 组合套餐"减"类型显示 `减N`
- 货币说明统一在 Footer 标注：`货币单位为人民币 (RMB)`
